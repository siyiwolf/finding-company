## Logo Design Concept (for professional designer)

### Core Elements:
1. **Symbol**: 融合"探索"与"科技"的抽象图形
   - **方案A**: 指南针 + 电路板的结合
     - 指南针代表"Finding"（发现、探索）
     - 电路线条代表智能硬件科技感
     - 整体呈圆形，象征完整、闭环体验
   
   - **方案B**: 山峰 + 信号波
     - 山峰代表户外运动
     - 向上发射的信号波代表智能、连接
     - 线条简洁现代
   
   - **方案C**: 字母\"F\"变形
     - F 同时像箭头（前进、发现）
     - 右上角加上一个小圆点（像GPS坐标）
     - 简约几何风格

### 4. **Logo Design - Version 4: Compass + Smart Network**
   - **核心理念**：指南针的精准定位 + 智能硬件的网络连接
   - **图形结构**：
     - 指南针外框（圆形）+ 内部十字北针
     - 北针指针由**电路节点和连线**构成（智能网络）
     - 十字交叉点是一个**发光节点**（GPS/定位核心）
     - 整体为单线风格（stroke-only），无填充
   - **布局优化**：
     ```
     [指南针图形]  Finding Company
                 Make Life Easy
     ```
     - 图形占左侧 1/3，文字右侧 2/3
     - 图形高度 = 文字x高度
   - **线条粗细**：主线条 2-3pt，节点圆点 4pt
   - **间距比例**：图形与文字间距 = 图形宽度的 0.3 倍

2. **Typography**:
   - 主体字体：无衬线字体（如 Inter, SF Pro, 苹方）
   - "Finding" 加粗大写，Company 小写常规
   - 或者全大写 "FINDING COMPANY" 显厚重感
   - 字母间距适中，体现专业与亲和平衡

3. **Color Palette (V4 Optimized)**:
   - **主色**: 科技蓝 **#007AFF** (首选) 或 **#4A90D9** - 明亮、现代、代表智能与可靠
   - **辅色**: 活力橙 **#FF9500** (更鲜艳) 或 **#F97316** - 户外、活力、易用
   - **中性色**: 深灰 (#1F2937) / 白 (#FFFFFF)
   - **发光节点**: 纯白 #FFFFFF 外围带浅蓝光晕 (#007AFF 20% opacity)
   - **渐变方案**: 蓝→橙(40%) 或 蓝→青(70%)
   - **重要**: 避免使用深蓝 (#2563EB)，优先选择明亮科技蓝 (#007AFF)

4. **Tagline 位置**:
   - 方案1: 主标下方 "Make Life Easy"
   - 方案2: 主标右侧 "Outdoor Intelligence"
   - 字体更细、更小

5. **Logo 结构**:
   ```
   [图形]  Finding Company
           Make Life Easy
   ```
   或
   ```
       Finding Company
       Make Life Easy
   [图形]
   ```

6. **风格关键词**:
   - 简约、现代、专业
   - 智能感但不冰冷
   - 户外运动基因（活力、探索）
   - 易用性（Make Life Easy）

### 应用场景建议:
- **产品本体**:  Smaller, monochrome version
- **官网/DTC**: Full color, large
- **App Icon**: Graphic only, square
- **社交媒体**: Horizontal lockup (graphic left, text right)
